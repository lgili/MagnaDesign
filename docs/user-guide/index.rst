User guide
==========

This section walks through the MagnaDesign desktop app end-to-end —
from launching the application to exporting customer-ready
deliverables. Each chapter focuses on one workflow step, with
annotated screenshots showing what to click, what to read, and
when each numeric output matters.

If you prefer the headless / batch workflow, jump to
:doc:`/user-guide/09-headless-cli`. If you'd rather start by
opening the example projects and reading the source, see
:doc:`/getting-started/first-design`.

.. note::

   Screenshots in this guide are auto-generated by
   ``scripts/generate_docs_screenshots.py`` against the
   ``examples/600W_boost_reference.pfc`` and
   ``examples/line_reactor_600W.pfc`` fixtures. Re-run the
   generator after a UI change to keep the documentation in sync.
